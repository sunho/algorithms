    Let x be price change in thousands.
    And f(x) be the total benefit.

    $ f(x) = (x + 30)(-2x + 60) - 10(-2x + 60) $

    $ = -2x^2 + 1800 + 20x - 600 $

    $ = -2x^2 + 20x + 1200 $
    
    $ = -2(x^2 - 10x - 600) $

    $ = -2((x-5)^2 -600 -25) $

    $ = -2(x-5)^2 + 1250 $

    The function will have the maximum 1250 when x = 5.

    Therefore 35000.
